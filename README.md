# University Planning & Productivity System

## Files Included:
- `dashboard.html` → Open in browser for full visual system
- `study.csv` → Import to Excel/Google Sheets
- `projects.csv` → Import to Excel/Google Sheets
- `skills.csv` → Import to Excel/Google Sheets
- `weekly.csv` → Weekly overview dashboard

## How to use in Google Sheets:
1. Open Google Sheets
2. File → Import → Upload each CSV
3. Place each on a separate sheet tab

## How to use in Excel:
1. Open Excel
2. Data → From Text/CSV → Select file
3. Repeat for each CSV on separate sheets
